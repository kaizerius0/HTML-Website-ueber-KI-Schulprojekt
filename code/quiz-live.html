Quiz – Live-Modus

Starte eine Quiz-Session und teile den Code mit deinen Schülern. Sie können während deines Vortrags am Quiz teilnehmen, du siehst die Ergebnisse live.

Eine neue Session erzeugt einen 4-stelligen Code. Die Schüler öffnen die Quiz-Seite, tragen den Code ein und spielen das Quiz durch. Sobald jemand fertig ist, erscheint das Ergebnis hier auf dieser Seite.

Die globale Bestenliste auf der normalen Quiz-Seite wird durch Session-Eintraege nicht beeinflusst.

// SESSION-CODE
----
TEILNEHMER: 0
Noch keine Eintraege. Warte auf erste Teilnahmen…
# NAME SCORE % DAUER
// SESSION BEENDET
----

// VOLLSTAENDIGE RANGLISTE

Keine Teilnahmen in dieser Session.
# NAME SCORE % DAUER

// KLASSEN-STATISTIK

// SCHWIERIGKEITSANALYSE – schwerste zuerst

Pro Frage: Anteil der richtigen Antworten. Niedrig = Thema noch nicht klar.
